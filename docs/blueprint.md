# **App Name**: CampusConnect

## Core Features:

- Home Feed: Teachers post announcements, events, and updates with real-time like, share, and comment functionality. Teachers (and only teachers) can post.
- Community Chat: Separate chat communities based on department, course, and branch, accessible to relevant students and teachers.
- NSS/NCC/Sports Registration: Students apply via digital form; designated mentors review applications with single-click accept/deny functionality. Accepted students get added to private community chat.
- Notes Repository: Mentors upload notes in PDF format. A search bar is available for users to search for notes across the entire college.
- Schedule Viewer: Students and teachers view timetables, attendance, and exam results by selecting their branch, year, class, and section from dropdown menus.
- Student Clubs: Students propose clubs/tutoring; mentors approve; community chat created for each club.
- Canteen Pre-booking: Canteen staff updates the menu with item name, price, and current stock. Students select items/quantities to book. Unique QR code is generated for payment/pickup.

## Style Guidelines:

- Primary color: Soft rose (#F4B4C5) for a comforting and inviting feel.
- Background color: Off-white (#FAF9F6) for a clean and light interface.
- Accent color: Pale lavender (#D6CADD) to create subtle highlights and call-to-action elements.
- Body and headline font: 'PT Sans' (sans-serif) for a friendly and readable feel throughout the application.
- Use minimalistic and consistent icons across all modules.
- Ensure a responsive and mobile-friendly design using Tailwind CSS grid and flexbox.
- Incorporate subtle transitions and animations for interactive elements, such as loading spinners or button hover effects.
